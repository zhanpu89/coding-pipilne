# 测试用例文档模板

每个模块独立文件：`doc/tester/{模块名}_测试用例.md`

**头部：** TC-YYYYMMDD-NNN / 版本 / 状态(🟡→🟢) / 日期 / 作者 / 关联文档

**用例表：**

| ID | 类型 | 标题 | 优先级 | 前置条件 | 测试步骤 | 预期结果 |
|----|------|------|--------|---------|---------|---------|
| TC-{模块}-{类型}-{序号} | UNIT/INTG/SEC/PERF/FE | ... | P0-P3 | ... | ... | ... |

用例 ID 规则：模块缩写(2-4字) + 类型缩写 + 序号。P0 标记"阻断"。每轮测试有结论节。
