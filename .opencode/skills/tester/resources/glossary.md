# 术语表（tester）

## 文档命名
| 产出物 | 格式 |
|--------|------|
| 用例文档 | `{模块名}_测试用例.md` |
| 测试报告首轮 | `{模块名}_测试报告.md` |
| 测试报告第N轮 | `{模块名}_测试报告_RN.md` |
| 进度追踪 | `_PROGRESS.md` |

## 测试结论
| 结论 | 条件 |
|------|------|
| ✅ 通过 | 无 P0/P1，P2 ≤ 3，PASS ≥ 90% |
| ⚠️ 有条件通过 | 无 P0，PASS ≥ 80%（3工作日修复 P1） |
| ❌ 不通过 | 存在 P0 或 PASS < 80% |
